# 자바스크립트 객체 (Object) 가이드

객체는 원시 값을 제외한 나머지 값(함수, 배열, 정규 표현식 등)을 의미하며, 자바스크립트를 구성하는 가장 기본적인 단위입니다.

---

## 1. 객체의 정의와 구성

객체는 0개 이상의 **프로퍼티**로 구성된 집합입니다.

- **프로퍼티:** 키(Key)와 값(Value)으로 구성됩니다.
- **메서드:** 프로퍼티 값이 함수일 경우, 이를 일반 함수와 구분하여 메서드라 부릅니다.

```javascript
const user = {
  name: "철수", // 프로퍼티
  sayHello: function () {
    // 메서드
    return `안녕하세요, ${this.name}`;
  },
};

console.log(user.sayHello()); // 안녕하세요, 철수
```

---

## 2. 객체 생성 방법

자바스크립트는 다음과 같은 다양한 객체 생성 방식을 지원합니다.

- **객체 리터럴** (가장 일반적인 방식)
- `Object` 생성자 함수
- 생성자 함수
- `Object.create` 메서드
- 클래스 (ES6)

> **💡 객체 리터럴 특징:** 중괄호(`{}`) 내에 프로퍼티를 정의합니다. 이는 값으로 평가되는 표현식이므로 닫는 중괄호 뒤에 세미콜론(`;`)을 붙입니다.

---

## 3. 프로퍼티 (Property)

### 규칙 및 특징

- **프로퍼티 키:** 문자열(빈 문자열 포함) 또는 심벌 값. 식별자 네이밍 규칙을 따르면 따옴표 생략 가능.
- **프로퍼티 값:** 자바스크립트에서 사용할 수 있는 모든 값.

```javascript
var person = {
  firstName: "Ung-mo", // 따옴표 생략 가능
  "last-name": "Lee", // 네이밍 규칙 미준수 시 따옴표 필수
};
```

### 프로퍼티 특징 요약

1. **동적 생성:** 존재하지 않는 키에 값을 할당하면 런타임에 생성됩니다.
2. **중복 선언:** 동일한 키를 중복 선언하면 나중에 선언한 프로퍼티가 덮어씁니다.
3. **키의 타입 변환:** 문자열/심벌 외의 값을 사용하면 암묵적으로 문자열로 변환됩니다. (예: 숫자를 키로 쓰면 문자열 "0"이 됨)

---

## 4. 프로퍼티 접근 및 조작

### 접근 방법

1. **마침표 표기법 (`.`):** `person.name`
2. **대괄호 표기법 (`[...]`):** `person['name']` (반드시 따옴표로 감싼 문자열이어야 함)

### 갱신 / 생성 / 삭제

```javascript
const person = { name: "woong" };

// 갱신
person.name = "won";

// 동적 생성
person.age = 25;

// 삭제
delete person.age;
```

## 5. ES6 객체 리터럴 확장 기능

### 프로퍼티 축약 표현

변수 이름과 프로퍼티 키 이름이 동일할 때 생략 가능합니다.

```javascript
let x = 1,
  y = 2;
const obj = { x, y }; // { x: 1, y: 2 }
```

### 계산된 프로퍼티 이름

문자열로 평가되는 표현식을 대괄호로 묶어 키를 동적으로 생성할 수 있습니다.

```javascript
const prefix = "prop";
let i = 0;

const obj = {
  [`${prefix}-${++i}`]: i,
  [`${prefix}-${++i}`]: i,
}; // { 'prop-1': 1, 'prop-2': 2 }
```

### 메서드 축약 표현

`function` 키워드를 생략하고 정의할 수 있습니다.

```javascript
const obj = {
  name: "Lee",
  sayHi() {
    // 메서드 축약
    console.log("Hi! " + this.name);
  },
};
```
